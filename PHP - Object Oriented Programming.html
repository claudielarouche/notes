PHP - Object Oriented Programming

Object Oriented Programming in PHP

For a simple site, OOP adds unnecessary complexity
For a complex site, OOP adds necessary simplicity

Defining a class

class Person { //name must start with capital letter and be camelcase
...
}

Function to get a list of all the existing classes

$classes = get_declared_classes();
foreach($classes as $class) {
    echo $class ."
    ";
}

(there will be a lot of predefined PHP classes that have nothing to do with your code)

You can verify the existence of a class like this:
if(class_exists("Person"){
    ...
}

Methods

Methods are basically the same thing as functions but they are called "method" because they exist inside a class.

List of existing methods:
$methods = get_class_methods('Person');
foreach($methods as $method) {
    echo $method . "
    ";
}

Check if method exists:
if(method_exists('Person', 'say_hello')){
    ...
}

Executing a method:
$person = new Person();
$person->say_hello();

Verify the class of a variable:
if(is_a($person, 'Person')) {
    ...
}

Attributes (variables inside a class)

class Person {
    var $first_name; //outside class, you could have just said $first_name and it would be fine but in a class, you NEED the keyword var
    var $last_name;

    var $arm_count = 2;
    var $leg_count = 2;

    function say_hello() {
      echo "Hello from inside the class ". get_class($this) .".
      ";
    }
    function full_name() {
      return $this->first_name . " " . $this->last_name;
    }
}

echo $person->arm_count; //this is how you echo an object's attribute

To show all the variables of a class and their default values:
$vars = get_class_vars('Person');
foreach($vars as $var => $value) {
    echo "{$var}: {$value}
    ";
}

Check if a property exists:
echo property_exists('Person', 'first_name'); //will return 0 or 1
