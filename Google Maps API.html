Google Maps API

Google Maps API: http://storelocator.googlecode.com/git/index.html
Reference: http://storelocator.googlecode.com/git/reference.html

Step 1:
Create empty divs called map-canvas and panel. They remain empty, they will be filled by Javascript

Step 2:
Includes:

<script type="text/javascript" src="_scripts/jquery-1.8.2.min.js"></script> <!-- Javascript -->
<script type="text/javascript" src="http://maps.googleapis.com/maps/api/js?sensor=false"></script> <!-- Google API -->
<script type="text/javascript" src="_scripts/store-locator.compiled.js"></script> <!-- Store locator downloaded from Git (URL above) -->

Note: An excel exported CSV will not load into the store locator application properly because it needs to have the double-quote around all the values (which Excel will not do)

See Store locator under jQuery in Bluehost
