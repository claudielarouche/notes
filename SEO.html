Page Titles

Writing clear, concise, unique pages titles and descriptions that accurately describe Web content is the single most important thing you can do for search engine optimization.

The importance of tiles and descriptions cannot be overstated. They are the only thing that a searcher has in order to make an informed decision about the relevancy of their search results.

The average maximum number of characters external search engines, such as Google, Bing and Yahoo, display for titles is approximately 70 characters, including spaces.

Include the most relevant terms at the beginning of the page title

Make your page title unique: Unique page titles help the search engine understand that similar pages are distinct. They also help ensure that users do not need to sift through many pages that have the same name in order to find the information they need.

Page Description

Try to have the title and description support each other to enhance searchers ability to determine relevancy.

The average maximum number of characters external search engines, such as Google, Bing and Yahoo, display for descriptions is approximately 160 characters, including spaces.

Inverted pyramid style writing is widely used by journalists. Inverted pyramid style writing begins with the conclusion, expressed in a single sentence. The first or next paragraph of the text then goes on to give the next important points, leaving the details and supporting information in the rest of the article. The same technique should be used for writing descriptions.

Tell the user what they can expect to find in the Web page rather than explaining the programs or services. That is the job of the Web page. For example:

Use this: Information for authors on writing titles, descriptions and keywords for site optimation.
Instead of that: Writing clear, concise, unique pages titles and descriptions that accurately describes Web content is the single most important thing you can do for search engine optimization.

Keywords

Keywords are a contentious issue. Due to the misuse of the Keyword metadata field, external search engines do not use the keyword field for relevancy and ranking purposes. See the article Death of a Meta Tag for more information. http://searchenginewatch.com/2165061

However, site search engines can use the keyword metadata field for:
- helping to boost a Web page's relevancy ranking,
- spellings and variants, and
- cross-referencing.

Character length: Find out what the keyword metadata field character limit is for your site search engine. Some site search engines can read up to 400 characters, including spaces, in the metadata field.

Page content is the primary source for metadata keywords. Extract the keywords from the text and use them in the keyword field.

Content

Begin with the most important information

When organizing your content, use an inverted pyramid structure to guide your user to the information they need to
complete their task, as illustrated below.

1. Start each page with the most important idea, step or information.
2. Add details and tasks by order of importance.
    Use links to direct users to related information and tasks.
    Present links as bulleted items.
3. Include information for users who want to learn more or have an advanced knowledge of the topic or service.
4. Include related links

Writing for readability is one of the principals for content development on Canada.ca (see section 1.22). To assess if
a piece of content meets the requirements for readability, you can use one of two free tools available for either official
language:

- English – The Flesch-Kincaid tool is bundled within Microsoft Word.

To activate this tool, open Word and select File>Options>Proofing. Check the box labelled “Show readability statistics.” To test a portion of text, highlight the text. Select Review>Spelling & Grammar. When Word has finished checking the text, it will ask if you
want to check the rest of the content. Click No. The readability statistics dialog box will appear and show the Flesch-Kincaid reading level.

- French – www.scolarius.com

Rates text using a point system based on groupings of grade levels. For grades 6-8, the text should score at the high end of “primaire” or the low end of “secondaire”. To use this tool, open the web page and select the type of text you are measuring. Copy and paste your content into the field provided on the website. Click Soumettre. You will see a single score for your content.

If the text you are testing contains bulleted items, add a period to the end of each bullet for the tool to properly evaluate the text.

Do one or more of the following to improve the readability of your text:
- edit the text to shorten words, sentences and paragraphs
- replace any jargon with plain language terms known to users
- remove any extraneous detail that does not hinder the ability to complete the task
- replace secondary references to a program with “program” instead of the full name of the program or the acronym
- move key tasks to bulleted list items

Lists

A vertical list makes it easier for users to scan content. There are two types of lists used:
- Numbered lists, to show the ranking or priority of items or for step-by-step instructions where the lead-in states the number of points that follow
- Bulleted lists, which must:
    - begin with a lead-in paragraph that introduces or applies to all the list items or emphasizes the relationship between each element
    - include in the lead-in any word or phrase that is common to all the items
    - have only one idea in each bulleted item
Use a consistent or parallel grammatical structure. This means that each item begins with the same part of speech and uses the same verb tense, sentence or phrase structure, voice and perspective. When writing list items:
- either use all negatively or all positively phrased items
- use the same structure
    - For example, if you use the imperative (or command) mood in the first item of your list, use the imperative mood in each item. Similarly, if your first item begins with a noun, continue listing items that begin with nouns.
- specify “all” or “or” in the introductory paragraph

Tables

You can verify whether a table contains only relevant data by ensuring that:
- the value of each cell relates to the column and row headers
- the entries within a given column exclude information that could be considered a subhead
- the value of each cell under a column header is relevant only to the column header
- the value of each cell is aligned with the column header that appears directly above it

Introduce your table with a brief, descriptive and relevant heading that explains what information is in the table

Each column and row of a table must have column and row headers that describe the information contained in a particular column or row of a structured table.

Row and column “headers” differ from what this document refers to as “headings” or “subheadings.” Headings refer to the title of a table or a section of content that appears on a web page.

Your web team will use information in a table’s column and row headers to create the web page in a way that allows a screen reader to identify the value of a specific cell using the values of its column and row headers.

When formatting your table:
- Do not use only colour or texture to explain the meaning of table content
- Include words that explain the meaning of the cell’s colour or background “texture.”
- Avoid formatting cells with textures or dark colours; this can make text difficult to read.
- Avoid blank cells in tables
- Include words that explain why a cell may have no value. Examples may include:
    - “No data”
    - “0” (zero)
    - “N/A” (not applicable)
Links

Be strategic about how you place links on a web page and how many links you provide:
- include links that support the user’s task
- do not embed links within paragraphs or tables
- only provide links to related content when they are directly related to the topic or task on the current page
- link to original web content or documents posted by the authoritative source

Cross-linking to related information can help users find information that is useful to them more quickly. Linking to your
pages from within other pages may also help improve search engine rankings. However, if you include too many links
on a page, users may leave the page without reading important information.

Linked text must be descriptive enough so that users understand what they will access when they click on it. It is not required to have link text exactly match the destination page title. Link text inherits some context from the other content on the page. Ensure the main keywords of the link text match the page title to assure the reader they
are in the right place. The destination page title can have additional keywords

To ensure your links are descriptive and useful:
- use the initial part of or the entire title of the destination page
- do not use the same link text to point to two different pages will confuse users, especially people with disabilities who are using a screen reader
- do not use vague terms such as “click here” or “more”
- do not including prose or promotional messaging that might confuse a user who is trying to decide whether to navigate to the page
- when more than one hyperlink on a web page links to the same content, use the same hypertext text to link to that content
