HTML5

[camelCaseWordsEnabled: false]

Main features

- Video Support
- Audio support
- Canvas element
- Offline storage
- New form elements
- Drag and drop
- Geolocation
- Local storage

For a list of features you can use per browser, go to http://caniuse.com

Example

<video id="myVideo" controls preload="auto">
    <source src="videofile.mp4 type="video/mp4" />
    <source src="videofile.webm type="video/webm" />
    <source src="videofile.ogv type="video/webm" />
    <!-- if video is not supported -->
    <p>Your browser does not support HTML5 video.</p>
</video>

Data tag

Example:
<li class="vacation onsale" data-price='399.99'>

You use the data tag to provide additional data to elements on the page. You can then use jQuery to access that information.

.data(<name>)

You can SET the value by providing a name AND a value: .data(, )
