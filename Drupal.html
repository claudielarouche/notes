==Install Drupal locally using DAMP==

1. Download DAMP from acquia: https://www.acquia.com/downloads (Acquia Dev Desktop 2)
apache web server port: 8083
mysql database port: 33067

2. Go to https://www.drupal.org/project/drupal to download core Drupal (right now you have acquia Drupal)

3. Uncompress Drupal and rename the folder to the name of your site (I'll call mine drupalclaudie)

4. In Acquia Dev Desktop, click on the little + at the bottom of the window and select Import Local Site

5. Browse to your desktop and select drupalclaudie folder

6. Leave create a new database selected and call database drupalclaudie

7. Click OK

8. Access the website and you will be asked to continue Drupal's install

9. Select standard profile and click Save and Continue

10. Configure your site

All done, bravo!

Note: If installing on WAMP, you will have to duplicate a file called settings.php. DAMP takes care of that for you.
Also, Drupal requires 40 Mb of memory but WAMP typically gives it only 2, so there may be a setting somewhere you have to change

==Develop Themes==

Note: If you manually download modules, you have to place them in sites > all > modules. If this does not exist, create it.

1. Download the Devel module and the Theme Developer module
2. Enable them but make sure you disable them if ever your website becomes public facing

There is a module called condition style sheet which allows you to render your site differently based on the browser used

In your theme.info file, you would then type something like this:
stylesheets[all][] = not-ie.css
; Set the conditional stylesheets that are processed by IE.
conditional-stylesheets[if IE][all][] = for-ie.css

More details about what to put in theme.info file: https://www.drupal.org/node/171205
