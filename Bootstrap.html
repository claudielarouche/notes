Step 1: Create a new folder somewhere for your new project

Step 2: Follow instructions below to download Bootstrap

Download Bootstrap from github

1. Go to http://getbootstrap.com/ and click on github at the bottom of the screen (which should bring you here: https://github.com/twbs/bootstrap)
2. Click on the download ZIP button. Downloading the github version should ensure you have the latest bootstrap version, also you will have more content
3. Unzip bootstrap
4. In your project folder, create two folders, one called "_" and one called "images"
5. Inside _ create a folder called components
6. Copy the fonts folder to _ folder
7. Copy the js and less folders to the components folder
8. Open the js folder and get rid of the test folder
9. Inside _ folder, create two folders: css and js. In these folders, we will be put the compressed versions of the LESS and js documents
10. Download jquery (bootstrap needs jquery to work)at http://jquery.com/ You can download uncompressed since we will be compressing all javascript files later
11. Rename the jquery file into jquery.js

Compressing files

1. Create another file in the folder _/components/less called _mystyles.less
2. Create another file in the folder _/components/js called _myscript.js
3. Go to grunt.txt in the documentation and follow the steps to get grunt going

The following steps are to be done in cmd:

4. Type grunt less:lessbootstrap (this will compress the bootstrap files)
5. Type grunt cssmin (this will minify the bootstrap css)
6. Type grunt concat (this will concatenate all javascript files)
7. Type grunt uglify:uglifybootstrap (this will make the bootstrap.js file all on one line)
8. Type grunt (now your grunt is in wait mode and will make necessary changes as soon as _myscript.js or _mystyles.less are updated

Now you can go to bootstrap's website (http://www.getboostrap.com) and under getting started, you can grab the starting HTML code. You will have to update the link to the bootstrap location properly though.

Building the structure of the page

It's a good idea to give an ID to each page so that they are easy to target in CSS
<body id="home">

<section class="container">
    <div class="content row">
      <section class="main col col-lg-8">
    </section> <!-- main-->

    <section class="sidebar col col-lg-4">
    </section><!-- sidebar-->
    </div> <!-- content-->
</section> <!-- container -->

Row is the technical term that you use in Bootstrap to define each row of your document. Content is a class that I'm defining to identify the main section of the code that's going to allow us to center this main layout on the page.

The two sections (col-lg-8 and col-lg-4) they have to add up to 12

I like to alternate between divs and sections to make things easier to read

Now that we have the structure in place, we can copy paste index.php to create all our other documents (about.php etc. etc.)

Other options:
- If you add a class of navbar-fixed-top to the top div, it will remain at the top all the time.
- If you add a class of active to one of the li element, it will mark it as active
- If you add a class of navbar-inverse to the top div, the navbar will become black
- the div at the top can be a div or section, it doesn't matter

To add a divider:
<li class="divider"></li>

To make the dropdown drop down automatically, add the following code to _myscript.js:
$(function(){
    //Make menu drop down on hover

    $('ul.nav li.dropdown').hover(function(){
      $('.dropdown-menu', this).fadeIn();
    }, function(){
      $('.dropdown-menu', this).fadeOut('fast');
    }); //hover
}); //on jQuery loads

Bootstrap allows you to make certain things visible only when the site is visited by certain device sizes: http://getbootstrap.com/css/#responsive-utilities

Example, this paragraph will not appear on a phone: <p class="hidden-xs">

Buttons: http://getbootstrap.com/css/#buttons
Example:
<a href="artists.php" class="btn btn-primary">About the artists</a>

Tooltips: http://getbootstrap.com/javascript/#tooltips
Example:
<a href="artists.php#Barot_Bellingham"><img data-toggle="tooltip" data-original-title="Barot Bellingham" src="images/artists/Barot_Bellingham_tn.jpg" alt="Photo of Barot Bellingham"></a>

Must add this line to _myscript.js: $('[data-toggle: tooltip]').tooltip({animation: true})

Media indent (image next to the text): http://getbootstrap.com/components/#media
media-left must be placed BEFORE media-body, media-right must be placed AFTER media-body

Example:
<article class="aboutthevenue media">
    <h2>About the venue</h2>
    <div class="media-body">
      <p>Paragraph 1</p>
      <p>Paragraph 2</p>
    </div>
    <a class="media-right" href="venuetravel.php">
      <img src="images/hotels/contempo.jpg" alt="Photo of Hotel Contempo">
    </a>
</article>
