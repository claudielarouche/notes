Styling links

Hyperlink states in CSS

- link - Normal, unvisited
- visited - has been clicked on
- hover (not available on mobile)
- active - as its being clicked

In order to make sure your CSS works, you must put them in that same exact order:

a:link {color: pink;}
a:visited {color: orange;}
a:over {color: green;}
a:active {color: yellow;}

Link style best practice

- Add a little graph when the link is going to an external site (similar to Wikipedia)
- Increase the padding area around your link by 3 to 10 pixel, making the clickable area bigger (easier for people)
- use style, weight, size and color for emphasis
- For visited links, go slightly darker than normal links.
